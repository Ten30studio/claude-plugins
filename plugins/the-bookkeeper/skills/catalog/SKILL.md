# Market-Rate Price Catalog

When the user asks "what should I charge for X" or "what's the market rate for Y" or when computing the value of a deliverable for a statement, reference this catalog. The Bookkeeper is honest about market rates — not aspirational, not deflationary. **Mid-point of the typical range, with source citations.**

## How to use this catalog

1. **Identify the service category** the user is asking about.
2. **Locate it in the catalog below** (or the closest analog if not exact).
3. **Quote the mid-rate** as the default Ten30/agency-style price.
4. **Cite the source range** — *"$X–$Y per [Upwork Pro / Fiverr Pro / industry survey], mid-point applied."*
5. If the user asks for **higher-end pricing** (boutique, enterprise), use the high end of the range.
6. If the user asks for **lower-end pricing** (DIY-adjacent, emerging market), use the low end of the range.

If a service isn't in the catalog, do not guess silently. Tell the user it's not in the catalog and offer to research market rates by category.

## Refresh policy

The catalog should be refreshed roughly quarterly. Source ranges drift over time. When refreshing:
- Re-check Upwork Pro / Fiverr Pro / 99designs landing-page rates
- Re-check small-business consultant rate surveys (BLS, Indeed, BizFluent)
- Re-check copywriting freelancer rates (AWAI, ProBlogger)
- Note the refresh date at the top of the body

## What this catalog does NOT do

- Does not negotiate. The user picks the rate; the catalog provides reference.
- Does not collect payment. The Bookkeeper records value — payment processing is upstream.
- Does not advise on tax, legal, or compliance — those go through The Counsel pattern or human counsel.

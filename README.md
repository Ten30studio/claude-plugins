# Ten30 Studio · Claude Code Plugins

> Plugins built by [Ten30 Studio](https://ten30studio.com) for AI-native production houses, productized agencies, and chat-first SMB operators.

Each plugin in this marketplace packages a real operational pattern from the **[Kitt agent stack](https://github.com/Ten30studio)** — the autonomous AI partner that runs Ten30 Studio's day-to-day operations. We've open-sourced the patterns we use ourselves so other operators can adopt them.

## Installation

```bash
# Add this marketplace
/plugin marketplace add Ten30studio/claude-plugins

# Discover what's available
/plugin discover

# Install a specific plugin
/plugin install the-bookkeeper@ten30studio
```

## Available plugins

### 📒 The Bookkeeper

Track the market-rate value of every service you deliver — even (especially) the work you give away.

- Itemized service-delivery statements with sourced market-rate pricing
- Courtesy discounts visible (make the gift visible)
- Client-by-client engagement history
- Portfolio view across all clients
- All data stays local on your machine

For productized agencies, solo consultants, freelancers, AI-native operators.

[See plugin →](./plugins/the-bookkeeper/)

## Philosophy

We believe:

1. **AI-native production houses are about to eat the long tail of agency services**, the way Shopify ate the long tail of e-commerce.
2. **The operators who win the next decade will be the ones who package their internal patterns** as reusable, distributable, and (where possible) open-source plugins.
3. **The plugin ecosystem itself is the leverage** — every solo founder who installs your plugin becomes part of your distribution network.

Ten30 Studio is building under this thesis. These plugins are our public proof of work.
